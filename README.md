# TimeTravel Agency - Webapp Interactive

Webapp pour une agence de voyage temporel fictive, créée avec IA générative.

## 🛠️ Stack Technique

* **Frontend** : React, Vite, Tailwind CSS, shadcn/ui
* **Animations** : Framer Motion, Lucide React
* **Intelligence Artificielle** : Mistral AI API (via Edge Functions / client-side integration)
* **Hébergement** : Lovable / Vercel

## ✨ Features

* **Landing Page Interactive** : Immersion immédiate avec un design moderne, futuriste et des animations fluides.
* **Galerie de 3 Destinations Temporelles** : Exploration immersive de trois époques clés avec descriptions et visuels dédiés.
* **Chatbot IA Conversationnel** : Agent intelligent intégré pour répondre aux questions des voyageurs temporels en temps réel.
* **Recommandations Personnalisées** : Algorithme ou questionnaire dynamique suggérant la meilleure époque en fonction du profil utilisateur.

## 🤖 IA Utilisées

* **Génération de Code** : Lovable / Bolt.new (propulsé par Claude 3.5 Sonnet) pour l'architecture et les composants.
* **Moteur du Chatbot** : Mistral Small / Large via API pour une expérience conversationnelle fluide et locale.
* **Création de Visuels** : Midjourney pour les concepts visuels fixes et Runway pour les animations vidéo (Projet 1).
